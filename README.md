# 🤖 Chatbot RAG avec FastAPI & Supabase

Ce projet implémente un chatbot intelligent basé sur une architecture RAG (Retrieval-Augmented Generation).  
Il permet de répondre aux questions des utilisateurs à partir de contenus personnalisés (FAQ et documents).

---

## 🚀 Fonctionnalités

- ✅ Création de chatbots
- ✅ Ajout de FAQ
- ✅ Upload de documents (PDF / PPT)
- ✅ Extraction automatique du texte
- ✅ Chunking et vectorisation (embeddings)
- ✅ Recherche sémantique avec Supabase (pgvector)
- ✅ Réponses basées sur le contenu (RAG)
- ✅ Filtrage des réponses hors contexte (threshold)
